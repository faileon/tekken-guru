Welcome to Tekken.guru, a web application containing move lists of all the characters from the game, including a video of every move and frame data with the ability to apply a variety of filters to find the move you are looking for. Along with the move lists you can find recommended punishes, key moves, combos and a written overview for each character.

The app should work both on desktop and mobile, including in offline. For best mobile experience, don't forget to install the actual PWA with the "Add to Home Screen" button.

I am more or less done with the core of the app and now the main focus is to provide all the data, but there are still a few things on my mind that I would like to slowly implement.

  1. Ability to add moves to favorites - dilemma between offline and online storage. Pros and cons for each.
  2. Sorting. (by damage, frames, hits, ...)
  3. Filtering by hit level - ability to toggle between all hits vs at least one (currently only the latter is implemented).
  4. Ability to filter throws.
  5. Ability to copy link for specific move. (e.g. characters/leroy/movelist/#002)
  6. Input notation display settings - tired of playstation layout?
